# Table

Table component. Loosely inspired by the react-table library. The component supports filtering, sorting, custom cell components, and extra dropdown actions per row.
 
## Properties
<Props of={Table} />

## Basic usage

### Normal table
<Playground>
<Table
    columns={mockTableData.columns}
    data={mockTableData.data}
/>
</Playground>

### Table with title and search and options
<Playground>
<Table
    title="Table title"
    hasSearch={true}
    columns={mockTableData.columns}
    data={mockTableData.data}
    options={[{
        id: 'test',
        name: 'Test Option',
        onClick: () => alert('You clicked me!')
    }]}
/>
</Playground>


### Table with custom cells
<Playground>
<Table
    columns={mockTableData.columns}
    data={mockTableData.data}
    customComponents={{
        cell: (value) => <button>{value}</button>
    }}
/>
</Playground>
