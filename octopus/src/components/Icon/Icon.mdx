# Icon

Icon components. These correspond to the svgs in `src/static/icons`. The component uses CSS to modify height, width, stroke color and fill color. So the SVG asset should have an `icon-container` class on the root SVG tag, and `fill-color` and `stroke-color` classes wherever you would like stroke or fill color to be customized. Fill color and stroke color names should be a [a valid style color](/styles/colors)
